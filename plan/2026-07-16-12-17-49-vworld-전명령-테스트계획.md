# vworld 전 명령 실사용 테스트 계획

목적: `references/docs/USAGE.md`(전 명령 사용설명서)에 기재된 **모든 명령·서브커맨드가 실제로 동작하는지** 실측 검증한다.

- 바이너리: `target/release/vworld` (v0.2.0, 2026-07-16 재빌드본)
- 판정 기준: exit 0 + (`"ok":true` 또는 정상 표 출력 또는 산출 파일 생성·크기/내용 확인). 에러 처리 케이스는 "명확한 에러 메시지 + exit≠0".
- 원칙: 사용자 실 설정(`~/.vworld/config.toml`)을 변경하는 테스트(config add/remove)는 **임시 `--config` 파일**로 격리. 파일 산출물은 스크래치패드에 생성.
- 무거운 호출(`ned --all`, `--by-hjd`)은 소형 법정동으로 제한.
